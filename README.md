# Deepak Krishna Portfolio

Professional portfolio website for Deepak Krishna, an Automotive Test Engineer with 9+ years of experience in embedded systems testing, HIL/SIL validation, automation, and automotive diagnostic protocols.

## Overview

This repository contains a single-page portfolio designed to present professional experience, education, technical skills, certifications, and contact information in a clean, modern, and recruiter-friendly format.

## Highlights

- Responsive single-page portfolio layout
- Professional hero section with animated visual treatment
- Work experience timeline for automotive testing roles
- Education, skills, and certifications sections
- Branded favicon and polished icon system
- Mobile-friendly navigation and interactions

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- SVG icons and favicon assets

## Run Locally

Use any simple static server. For example:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## What The Site Includes

- About summary
- Professional experience timeline
- Education background
- Technical skills overview
- Certifications showcase
- Contact section with direct links

## Customization

If you want to adapt this portfolio further, the main content and styling are in `index.html`.

Typical updates include:

- Replacing profile or company images
- Updating certifications and experience
- Adjusting colors, icons, and layout
- Enabling GitHub Pages or another static hosting provider

## Repository Purpose

This repository is intended to serve as the source code for a personal professional portfolio and as a presentable GitHub project for recruiters, hiring managers, and collaborators.
